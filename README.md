# 📃 Hermes Project
## 프로젝트 소개
- 기존 웹 사이트보다 웹 접근성을 좀 더 높이고자 웹 사이트 재구축하기 위함이 목적
- 메인, 로그인, 장바구니, 회사소개 페이지 제작
- 320px, 768px, 1280px, 1440px로 반응형 웹 사이트로 제작
## 사용 기술
- HTML
- SCSS
- JavaScript
## 보완 사항
- 제작되지 못한 페이지 구성 예정
- 추가적인 JavaScript 기능 구현 필요
